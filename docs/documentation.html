Documentation

The biggest feature of the class file is the content command. This command takes four required arguments: 

  
    \content{
      main
    }{
      presentation
    }{
      nonpresentation
    }{
      comments
    }

The main argument can be anything you want displayed in both versions of the file (theorems, the statements for examples and problems, etc.).

The presentation argument is anything that should be visible in the skeleton lecture notes, but not visible in the nonpresentation version of the file (blank space for working examples for example). If the presentation option is passed to the class, this and the main will be shown in the first of two columns, which by default is 7.25 inches wide (the page layout is landscape).

The nonpresentation argument is similar to the presentation argument, only that this is only visible in the nonpresentation version of the file. This could be used for solutions to examples, proofs of theorems, etc.

The comments argument is displayed in the second column in the presentation version of the file, which by default is 2.25 inches wide.
