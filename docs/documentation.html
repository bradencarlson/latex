Notes Class Documentation

Content

\content{main}{pres}{nopres}{comment}

Mandatory parameters:

  1. main - content to be shown in both versions of the file.
  2. pres - content to be shown in the presentation version of the file only.
  3. nopres - content to be shown in the non-presentation version of the file only.
  4. comment - comments to be shown to the right of pres in the presentation version of the file.
Each of these may be any LaTeX commands you wish. They are all defined using the \long keyword, meaning that the content of each of these is permitted to contain more than one paragraph. This is particularly useful for the nopres option, as this often contains solutions to problems which can be quite long.

Note:
If the comment parameter is nonempty, the class will draw a box around it, even if the comment parameter is just blank space. To avoid boxes where you do not want them, it is recommended to either use the % operator in your LaTeX code immediately after the opening brace of the comment parameter, followed on the next line by the closing brace, or to simply leave the comment parameter completely empty like so: {}.

Phantom

\phantom{pres}{nopres}

Mandatory Parameters:

  1. pres - content to be shown when the presentation option is specified.
  2. nopres - content to be shwon when the nopresentation option is specified.
This command is defined using the \long command, so each of the parameters may be as long as needed. This command was included for backwards compatibility in a previous version of the notes class.
