Notes Class Documentation

Content

\content{main}{pres}{nopres}{comment}

Mandatory parameters:

  1. main - content to be shown in both versions of the file.
  2. pres - content to be shown in the presentation version of the file only.
  3. nopres - content to be shown in the non-presentation version of the file only.
  4. comment - comments to be shown to the right of pres in the presentation version of the file.
Each of these may be any LaTeX commands you wish. They are all defined using the \long keyword, meaning that the content of each of these is permitted to contain more than one paragraph. This is particularly useful for the nopres option, as this often contains solutions to problems which can be quite long.

Note:
If the comment parameter is nonempty, the class will draw a box around it, even if the comment parameter is just blank space. To avoid boxes where you do not want them, it is recommended to either use the % operator in your LaTeX code immediately after the opening brace of the comment parameter, followed on the next line by the closing brace, or to simply leave the comment parameter completely empty like so: {}.

Phantom

\phantom{pres}{nopres}

Mandatory Parameters:

  1. pres - content to be shown when the presentation option is specified.
  2. nopres - content to be shown when the nopresentation option is specified.
This command is defined using the \long command, so each of the parameters may be as long as needed. This command was included for backwards compatibility in a previous version of the notes class.

Mathcommands Package Documentation

Commands

supp

\supp

Provides the \supp command for the support of a vector or function.

la and ra

\la and \ra

Maps the command \la to \left\langle, and provides a similar command for \right\rangle.

F, C, R, Q, Z, N, and N0

\F, \C, \R, \Q, \Z, \N, and \Nzero

Provides commands for standard sets in mathematics: the complex, real, rational, and natural numbers, as well as the integers. The set of natural numbers and zero is also defined.

tr

\tr

Provides command for the trace of a matrix. This command does not provide delimiters of any kind, it simply prints the letters tr.

det

\det

Provides command for the determinant of a matrix. This command does not provide delimiters of any kind, it simply prints the letters det.

Im

\Im

Provides command for the image of a map. This command does not provide delimiters of any kind, it simply prints the letters Im.

Aut

\aut

Provides command for the automorphism group of an object. This command does not provide delimiters of any kind, it simply prints the letters Aut.

Inn

\inn

Provides command for the inner automorphism group of an object. This command does not provide delimiters of any kind, it simply prints the letters Inn.

Gal

\gal

Provides command for the Galois group of an object. This command does not provide delimiters of any kind, it simply prints the letters Gal.

lcm

\lcm

Provides command for the least common multiple of two numbers. This command does not provide delimiters of any kind, it simply prints the letters lcm.

mod

\mod{n}

This commands prints out (mod n). For example, one might type 5 \mod{3}, which would be rendered as 5 (mod 3). This command does provide the parenthesis, but they are not resized to fit the content between them.

Environments

Theorem

\begin{theorem} ... \end{theorem}

Provides the theorem environment. If the current value of the subsection counter is greater than zero, the label for this theorem is set to \thesubsection.\thetheorem, otherwise it is set to \thesection.\thetheorem. The counter \thetheorem is defined by the package file, and is reset upon declaring a new subsection of the document.

\begin{lemma} ... \end{lemma}

See Theorem.

Proof

\begin{proof} ... \end{proof}

Provides the proof environment. Simply prints out "Proof." at the beginning and prints a black square for the Q.E.D. symbol at the end of the proof.
