content

Mandatory parameters:

  1. main - content to be shown in both versions of the file.
  2. pres - content to be shown in the presentation version of the file only.
  3. nopres - content to be shown in the nonpresentation version of the file only.
  4. comment - comments to be shown to the right of pres in the presentation version of the file.
Each of these may be any LaTeX commands you wish. They are all defined useing the \long keyword, meaning that the content of each of these is permitted to contain more than one paragraph. This is particularly useful for the nopres option, as this often contains solutions to problems which can be quite long.
