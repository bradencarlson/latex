Installation

Linux

I prefer the following setup:

  1. Create a directory to host all user installed class files.
    2. mkdir ~/.settings
    cd ~/.settings
    git clone https://github.com/bradencarlson/lecturenotes.git
  2. Create symlinks to TeX's directory
    3. ln -s ~/.settings/lecturenotes/notes.cls /usr/share/texmf/tex/latex/custom/notes.cls
    ln -s ~/.settings/lecturenotes/mathcommands.sty /usr/share/texmf/tex/latex/custom/mathcommands.sty
  3. Run texhash
    4. texhash
